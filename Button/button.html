Button

Buttons are the soul of an interactive Web Application. Buttons allow users to take actions, and make choices, with a single tap

Primary Button

Primary button can be created by adding btn, btn-primary and can add these class for the respective variants btn-solid, btn-outline, btn-icon, btn-icon-text, btn-link

Primary-solid Primary-outline Cart Button

Secondary Button

Secondary button can be created by adding btn, btn-secondary and can add these class for the respective variants btn-solid, btn-outline, btn-icon, btn-icon-text, btn-link

Primary-solid Primary-outline Cart Button

Floating Button

Floating button can be create by adding class btn ,btn-float.
