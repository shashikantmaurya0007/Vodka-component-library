Images

Images are the important part of webapplication. good images can change the look and feel of the application.

Resonsive Image

To make your image responsive add class img-responsive to the img tag.

Rounded edge Image

To make responsive rounded edge image add img-rounded to img class.

Circle Images

To make image in circle shape add class img-circle to img tag class
