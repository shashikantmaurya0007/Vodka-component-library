Badges

Badges are small circle placed at either top right or bottom right to the image or icon .It can be used to diplay the current staus of user or count of notification recieved.

Status Badge

status badge can be created by adding badge class to the div container and add avatar class and size class(large,medium,small) to the image and status-badge class to span and status have three variants online , offline , do-not-disturb for desire style.

Square Badge

square status badge can be created by adding badge class to the div container and add avatar class along with square and size class(large,medium,small) to the image and status-badge class to span and status have three variants online , offline , do-not-disturb for desire style.

Icons badge

To create icon badge create div container and add class badge and add icon within span tag and give class icon-badge to it and give class badge-up to span tag whose value will go above icon
