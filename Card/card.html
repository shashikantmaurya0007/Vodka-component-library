Card

card contains all the information related to product.if you have ecommerce site card contains the details of product like image,description,Buttons to add to cart and price of product

Basic Vertical Card

create a div container and give classcard,vertical .in a card container create two div with clsssimg-container,lower-card respectively.In img-container add your image.in lower-card create a header h3 and p with class header-description and another p tag for description and add class card-description to it.and create another container with class btn-container which will contain all the button in that. for understanding of button class go to Button documentation page.
