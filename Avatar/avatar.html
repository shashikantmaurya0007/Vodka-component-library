Avatar

Avatars are images that users can set as their profile picture. On GitHub, they're always going to be rounded squares. They can be custom photos, uploaded by users, or generated as Identicons as a placeholder.

Profile Avatar

Profile avatars can be created by adding class avatar in img tag.

Sizes Avatar

Size of the avatars can be changed by adding small|medium|large along with avatar class

Square Avatar

Adding square class to your existing avatar class will create square avatars.
