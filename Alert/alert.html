Alert

An alert is used to display message to the user to grab his attention.

Simple Alert

You can create alert by adding class alert along with other classes like primary,error,success,warning

Primary!This is a primary alert-check it now!
Error!This is a error alert-check it now!
Success!This is success alert check it now !
warning!This is warning alert check it now !

Outline Alert

Outline alerts can be created by adding class alert along with class outline-primary, outline-error, outline-success, outline-warning

Primary!This is a primary alert-check it now!
Error!This is a error alert-check it now!
Success!This is success alert check it now !
warning!This is warning alert check it now !
